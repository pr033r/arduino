Informace o aplikaci Developed by Adam Lasak


Aplikace komunikuje s Firebase RealTime Database od firmy Google. Odečtené hodnoty se analogově přenášejí do čipu ATmega328, který je převede do JSON datového formátu a následně se po sériové lince (115200 baudů) pošle do čipu ESP8266. ESP čip se mezitím připojí na lokální Wi-Fi síť. Poté se připojí na Firebase RealTime databázi a pomocí autentifikace získá UUID, pod kterým později zapisuje data do databáze (jednotliví klienti jsou pak indentifikovatelní podle UUID - tudíž je lze rozeznat). ESP8266 odposlouchává na sériové lince a přenesený JSON formát si zformátuje a následně pošle přímo do Firebase (UUID už má mezitím získané).
